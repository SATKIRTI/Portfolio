Project Overview

It is a game in which boy tries to save itself from the dragon by jumping over the dragon which comes in its way. The score is updated when boy saves himself from the dragon.And once the boy and dragon get collide with each other then the game will over and then you have to restart the game. The project will contain HTML, CSS and JavaScript files. The HTML file adds structure to the game followed by styling using CSS. JavaScript adds functionality to the game.

File structure: index.html,style.css,script.js
