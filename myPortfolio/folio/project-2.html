Face Recognition

This is a Smart Attendance System using face recognition.In this,you can take attendance of different students by recognizing their faces.And the attendance will mark in a database and it can be send via mail to the particular person.

I build this project in the Microsoft Engage 2022 at Microsoft.

Project Overview

Basically,it is a Smart Attendance System which take the attendance.I build this project by using Python and for GUI used Tkinter and used OpenCV for recognition.In this project,I learned how to send a database via email.In this,we can record attendance of several person and mark attendance in a database and send it via email.

After receiving database which shows the attendance,the person have clear idea of their presence and absence in the class/office.

I build this project in the Microsoft Engage 2022 at Microsoft
